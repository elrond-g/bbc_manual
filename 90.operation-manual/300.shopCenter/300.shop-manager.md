# 店铺管理

店铺管理包含了店铺配置，店铺装修等店铺初始化等的配置。

## 店铺配置

店铺设置包含显示店铺名称，店铺LOGO，店铺描述，设置后显示在店铺首页。

![](images/131.png)
 
在线客服

3.0.4版本中加入了在线客服功能，在店铺配置的店铺描述下方，设置QQ和旺旺号

![](images/132.png)

设置后在商城的店铺首页等位置限时在线客服

![](images/133.png)
